# 项目计划文档模板

## 文件结构

```
plan/
├── plan_01_总体计划.md
├── plan_02_[模块名].md       # 2级任务
├── plan_03_[子任务名].md     # 3级任务
├── plan_04_[子任务名].md     # 3级任务
└── ...（按执行顺序连续编号）
```

---

## 命名规范

- **格式**：`plan_XX_任务名.md`
- **编号**：从 01 开始连续递增，不跳号
- **排序原则**：
  - plan_01 必须是"总体计划"（1级）
  - 2级任务后紧跟其所有3级子任务
  - 按照依赖关系和执行顺序排列

---

## 层级关系标记

使用 YAML frontmatter：

```yaml
---
level: 1/2/3              # 层级
file_id: plan_XX          # 文件编号
parent: plan_XX           # 父任务编号
children: [plan_XX, ...]  # 子任务编号列表
status: pending           # 状态
created: YYYY-MM-DD HH:mm # 创建时间
estimated_time: XX分钟     # 预估耗时
---
```
